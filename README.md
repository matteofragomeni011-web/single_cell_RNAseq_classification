# Single Cell RNA-seq Classification

## Project Overview

This project aims to classify single cells grown in different media using pathway-level features derived from gene expression data.

## Objectives

* Transform gene expression data into pathway-level features
* Train a classification model to predict growth conditions
* Identify the most relevant biological pathways

## Methods

* Data preprocessing and normalization
* Pathway aggregation
* Machine learning (Random Forest)

## Technologies

* Python
* Scanpy
* Scikit-learn

## Data
Dataset from GEO: GSE165686
